Para executar o protótipo, é necessário fazer download ou copiar o repositório para a máquina local. 

Em seguida, basta abrir o arquivo index.html, localizado dentro da pasta nfe-mock, utilizando qualquer navegador moderno. Não é necessário rodar servidor nem instalar dependências; toda a navegação ocorre de forma estática.

Após abrir o arquivo, o usuário pode navegar normalmente pelas páginas. 

As alterações implementadas concentram-se exclusivamente nas seções utilizadas como exemplo no trabalho acadêmico. 

As páginas modificadas foram: 
o banner da página inicial; 
o protótipo do botão “Manifestação Destinatário” com seu novo modal; 
a área de Legislação, especificamente o menu Ajustes SINIEF ao selecionar qualquer ajuste; 
a página “Consultar Inutilização”, com destaque para o campo de CNPJ e sua validação; 
e a seção de Convênios, que recebeu melhorias de design, hierarquia visual e comportamento em estados de hover.
